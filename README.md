# web-player

## Requirements
* [Node.js](https://nodejs.org)
* [Gulp](http://gulpjs.com)

## Steps for running
1. Build the frontend scripts using browserify
```
gulp browserify
```
2. Run main.js using node.
```
node main.js
```

## Developer
You can run the developer task of gulp. It is the default task.
```
gulp
```
